# Atlas

Policy analysis tool for New America's Education Policy Program.

## General Info

* url: http://atlas.newamerica.org/
* server: ``node.js``.
* client: ``React.js``, ``react-router``, ``Backbone``.
* build: ``webpack``.
* authentication: ``passport.js`` with ``passport-google-oauth``.
* database: ``MongoDB``.
